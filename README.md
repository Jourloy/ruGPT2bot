### Required:
1. [Python 3.8+](https://www.python.org/downloads/)
____
### Install:
1. Download repository
2. Unzip anywhere
3. Change bot token in **src/cofing.py**
4. Open folder in cmd/terminal
5. Type `pip install -r requirements.txt`
6. Type `python src/main.py` for run
____
